spring boot 集成 Stripe (demo)

本文最后更新于 2025年6月26日 晚上

什么是 Stripe？

在当今的数字交易世界中，集成支付网关已成为开发人员的一项基本技能，无论你是在构建电子商务平台还是基于订阅的服务，几乎每个现代应用程序都需要可靠的支付解决方案，其中最受欢迎的选择之一是 Stripe，它是全球无数网站使用的值得信赖的领先支付网关。

先创建 dto, service, controller, config 等基础包。

创建 StripeService：现在它非常简单，要连接到 stripe，stripe 为你提供 API。所以它期望你传递调用 stripe API，对此它需要你输入一些产品名称是什么（productName），它的价格是多少（amount），需要的数量是多少（quantity），然后最后你想支付的货币（currency）。
所以这四个字段，stripe API 需要连接到支付网关。为此，首先用这些字段创建一个 dto。
