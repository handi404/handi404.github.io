先掌握：[[Roles and Permissions]]

添加 UserController，我们在里面添加更改密码的端点，因此此端点不应处于 auth 之下，因为 AuthenticationController 或此 /api/v1/auth 是公共的。

然后我们需要一个更改密码请求类型的对象（ChangePasswordRequest），因此它将保存更改密码的信息。
使用 UserService 实现 changePassword 方法。

创建 ChangePasswordRequest entity

请求更改密码所需的信息，
所以对于更改密码，我们需要做的是要求用户提供

  • currentPassword(当前密码)，我们需要这个当前密码来确保用户已经知道他的当前密码，并且他想要更改它
  • newPassword(新密码)，这是用户将输入的新密码
  • confirmationPassword(确认密码)，我们需要确保用户在更改密码时没有输入错误
