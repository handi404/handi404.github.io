编写SecurityFilterChain

如果你使用的是 REST API，那么表单就不适合你了。所以在 REST API 中你没有界面，通常会使用 Postman 之类的工具。

所以还有一种不同于基于表单的身份验证的用户身份验证方法，它被称为 Basic Authentication (基本身份验证)。
顾名思义，basic 就是借助用户名和密码对用户进行身份验证的基本方法

配置 spring boot 项目以启用 Basic Authentication
首先检查 defaultSecurityFilterChain 的代码：在 idea 中双击 shift 搜索 SpringBootWebSecurityConfiguration

要做的是只启用 HTTP 基本身份验证，我们不需要表单身份验证，所以要做的是需要编写自己的过滤器。
