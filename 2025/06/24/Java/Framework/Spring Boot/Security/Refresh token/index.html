刷新令牌允许客户端请求新的访问令牌，而无需用户在每次访问令牌过期时重新进行身份验证，从而提高应用程序的安全性和用户体验。

什么是 refresh token(刷新令牌)

refresh token 一种令牌，是用来刷新我们的 access token(访问令牌)，这有助于我们改进和提高应用程序的安全性。
可以降低令牌的可用性或有效性，每当我们的客户端或其他客户端需要新令牌时，你需要做的就是传递此刷新令牌并请求一个新的令牌。
限制或减少访问令牌的到期日期，并且刷新令牌可以使用一段时间比如说一周，一周后，用户需要再次进行身份验证以获取新的访问令牌以及新的刷新令牌。

好处：

  • 安全性，因此用户将拥有一个具有短有效期的访问令牌，然后他将使用刷新令牌来刷新该访问令牌
  • 撤销访问权限，例如，如果您想强制所有用户重新登录，需要做的就是撤销您的刷新令牌，访问令牌过期后，用户必须重新连接或重新登录。
